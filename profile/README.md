## 🚀 Overview
Not everyone who needs a bootable USB is a command-line expert, and BalenaEtcher for Windows is built with exactly that in mind. It is a free, open-source flasher that strips the process down to its essentials: point it at an image, pick your USB drive or SD card, and let it write and verify the result. What is BalenaEtcher's advantage over doing this manually? It removes the two biggest pain points — the fear of erasing the wrong disk and the uncertainty of whether the write actually worked. System drives are hidden by default so only removable media appears, and every flash is automatically checked against the source. With native support for ISO, IMG, and compressed ZIP and GZ files, BalenaEtcher keeps making bootable media about as simple as it can get, whether you're installing Linux, setting up a device, or building a rescue drive.

## ✨ Features
- **Minimal, focused design** — BalenaEtcher does one thing well: flash an image to a drive with as few steps as possible.
- **Automatic verification** — each write is checked against the original image so you can trust the media before you boot from it.
- **Drive protection** — internal disks stay hidden by default, so there's no risk of overwriting your system drive.
- **Flexible input formats** — flash ISO and IMG images, plus compressed ZIP and GZ files, without extra preparation.
- **USB and SD support** — a single, consistent workflow serves both USB sticks and SD cards for a broad range of uses.

## 🎯 Getting Started
1. Open BalenaEtcher and choose **Flash from file** to select your image.
2. Plug in your USB drive or SD card and confirm BalenaEtcher has the correct target selected.
3. Click **Flash** to write the image to the drive.
4. Let the automatic verification step confirm the write finished correctly.
5. Remove the drive safely and boot from it on your target device.

## ❓ FAQ
**Is BalenaEtcher free?**
Yes. BalenaEtcher is free, open-source software available to download and use at no cost.

**Which versions of Windows are supported?**
BalenaEtcher supports 64-bit Windows, primarily Windows 10 and Windows 11. A currently supported version is recommended.

**Is BalenaEtcher safe to use?**
Yes. It hides internal drives so you can't erase your system disk by mistake, and it verifies every write. For added safety, download BalenaEtcher from its official site and confirm your drive selection before flashing.

**Do I need to unpack compressed images first?**
No. BalenaEtcher can flash compressed ZIP and GZ files directly, in addition to standard ISO and IMG images.

**Is my data secure?**
BalenaEtcher runs entirely on your own computer and requires no account. Because flashing erases the selected USB drive or SD card, back up anything important on it beforehand.
